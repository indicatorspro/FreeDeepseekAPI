# DeepSeek → FreeDeepseekAPI (Extension)

Adds a DeepSeek account to the local FreeDeepseekAPI **with one click**.

## Installation

**Firefox**
1. Open `about:debugging#/runtime/this-firefox`
2. "Load Temporary Add-on" → select `manifest.json`

**Chrome / Edge**
1. Open `chrome://extensions`
2. Enable "Developer mode"
3. "Load unpacked" → select this folder

## Usage
1. Start FreeDeepseekAPI (port 9655)
2. Open `chat.deepseek.com` and log in
3. **Send any message** (e.g., `ok`)
4. Click extension icon → **「➕ Add to FreeDeepseekAPI」**

For multiple accounts, repeat from different browser profiles.
